# API for [E-commerce-nextjs](https://github.com/m1xa4kala/e-commerce-nextjs) and [E-commerce-admin](https://github.com/m1xa4kala/e-commerce-admin)

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
